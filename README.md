 Réplicas de Sites com HTML, CSS e Bootstrap

Bem-vindo ao repositório RéplicasSitesHtmlCssBootstrap!
Este projeto contém uma coleção de réplicas de sites famosos e exercícios de design, desenvolvidos com HTML5, CSS3 e Bootstrap, com o objetivo de praticar técnicas de layout, responsividade e estilos modernos.

 Objetivo do Projeto

Aprender e praticar HTML e CSS modernos

Explorar o framework Bootstrap para layouts responsivos

Criar réplicas visuais fiéis de sites existentes ou exercícios de design

Desenvolver habilidades em design web e atenção aos detalhes visuais

Preparar uma base para projetos mais complexos de front-end

 Como Executar os Sites Localmente

Clone o repositório:

git clone https://github.com/OseiasDias/RéplicasSitesHtmlCssBootstrap.git

Entre na pasta do projeto:

cd RéplicasSitesHtmlCssBootstrap

Abra qualquer arquivo .html em seu navegador favorito para visualizar a réplica do site.

Não é necessário servidor local, pois todos os sites são front-end puro.
